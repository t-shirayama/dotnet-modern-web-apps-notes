# 構成と環境

ASP.NET Core は `appsettings.json`、環境別設定、環境変数、Secret Manager、Azure App Configuration、Key Vault など複数の構成ソースを扱えます。

設定値はコードに埋め込まず、Options pattern で型として受け取ると扱いやすくなります。

```csharp
builder.Services.Configure<PaymentOptions>(
    builder.Configuration.GetSection("Payment"));
```

環境ごとの差分は、ソース管理に入れてよいものと入れてはいけないものを分けます。

| 種類 | 扱い |
| --- | --- |
| 機能フラグの既定値 | `appsettings.json` |
| 開発環境の接続文字列 | Secret Manager / local env |
| 本番 secret | Key Vault / App Service settings |
| 環境名 | `ASPNETCORE_ENVIRONMENT` |

構成設計は運用設計でもあります。デプロイ後に値を変えられるか、監査できるか、secret がログに出ないかを確認します。
