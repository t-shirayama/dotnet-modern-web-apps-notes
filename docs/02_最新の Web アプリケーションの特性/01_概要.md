# 概要

最新の Web アプリケーションは、いつでも利用でき、さまざまな端末からアクセスされ、需要の変化に耐え、セキュリティと変更容易性を保つことが期待されます。

ASP.NET Core は、この前提に合わせて設計されています。クロスプラットフォーム、軽量なホスティング、DI、テスト容易性、クラウドとの相性が主な強みです。

この章では、最新 Web アプリを次の特徴で整理します。

| 特徴 | ASP.NET Core での見方 |
| --- | --- |
| クラウドでホストしやすい | 小さいフットプリントとスケールしやすいホスティング |
| クロスプラットフォーム | Windows / Linux / macOS とコンテナー対応 |
| 疎結合 | DI と NuGet による必要な機能だけの組み合わせ |
| テスト容易性 | 単体テスト、統合テスト、`TestServer` |
| UI の柔軟性 | MVC、Razor Pages、Web API、SPA、Blazor |
