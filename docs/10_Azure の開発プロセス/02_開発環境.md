# 開発環境

ASP.NET Core は、Visual Studio、Visual Studio Code、`dotnet` CLI、任意のエディターで開発できます。

Visual Studio は、デバッグ、プロファイリング、テスト、Azure 発行などが統合されており、Windows 開発では強力です。Visual Studio Code と `dotnet` CLI は、軽量でクロスプラットフォームな開発に向いています。

重要なのは、チームで再現できる開発環境を作ることです。

| 項目 | そろえるもの |
| --- | --- |
| .NET SDK | `global.json` や README で明示 |
| Node.js | フロントエンドがある場合は `.nvmrc` |
| DB | Docker Compose / localdb / dev DB |
| Secret | Secret Manager / env vars |
| 起動手順 | README / scripts |

ローカルで動くことと Azure で動くことは別です。構成、接続文字列、HTTPS、認証 callback、CORS、ログ出力先などは環境ごとに確認します。
