# 概要

ASP.NET Core Web アプリも、最終的にはブラウザーで HTML、CSS、JavaScript を扱います。

クライアント側技術を理解すると、MVC / Razor Pages を使う場合でも、SPA を使う場合でも、どこに責務を置くべきか判断しやすくなります。

この章では、HTML、CSS、JavaScript、jQuery、SPA framework、Angular、React、Blazor を、役割と使いどころで整理します。

| 技術 | 主な役割 |
| --- | --- |
| HTML | 文書構造とフォーム |
| CSS | 見た目、レイアウト、レスポンシブ |
| JavaScript | ブラウザー上の動作 |
| jQuery | DOM 操作と AJAX の簡略化 |
| SPA framework | 状態管理、routing、component |
| Blazor | C# / Razor による対話的 UI |
